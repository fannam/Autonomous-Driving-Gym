# Notebooks

This directory now holds notebooks only.

- `workflows/`
  Jupyter notebooks that drive maintained AlphaZero CLI workflows end to end.
- `analysis/`
  Exploratory notebooks for inspecting saved outputs and training behavior.
  Includes `evaluate_self_play_data.ipynb` for maintained self-play shard analysis across meta-adversarial and adversarial formats.

Open Jupyter from the repository root so relative paths resolve cleanly.
